# Pizza 🍕 Ordering System - Windows Forms (C#)

A fun and interactive Pizza Ordering System built using C# Windows Forms.  
This application allows users to customize their pizza orders by selecting size, toppings, crust type, and where to eat. The total price is calculated automatically based on the selections, and users can reset the form or place an order with a single click.

---

## ✨ Features

- **Pizza Size Selection:** Small, Medium, or Large options.  
- **Toppings:** Choose multiple toppings such as Extra Cheese, Tomatoes, Mushrooms, Onions, Olives, and Green Peppers.  
- **Crust Type:** Thin or Thick crust options.  
- **Where to Eat:** Eat in or Take out selection.  
- **Real-Time Price Calculation:** Total price updates automatically based on selections.  
- **Interactive GUI:** Buttons and radio/check boxes for easy order customization.  
- **Reset & Order Buttons:** Clear selections or place the order with one click.  

---

## Files in the Project

- `PizzaOrderForm.cs` → Main form with GUI and order logic  
- `PizzaOrderForm.Designer.cs` → Windows Forms designer file  
- `Program.cs` → Application entry point  

---

## ▶️ How To Run

1. Open the solution in **Visual Studio**.  
2. Build the project (Ctrl + Shift + B).  
3. Run the application (F5).  
4. Customize your pizza using the form controls and place the order.
